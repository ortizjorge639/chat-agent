CONFIRM AZURE OPENAI ACCESS

Ensure you have an Azure OpenAI resource with a deployed LLM. If you're
using the same resources from the data extraction experiment, we can
reuse it. The chatbot only needs the endpoint, API key, and deployment
name. Any model that supports tool calling works (for example: GPT-5 mini).


VERIFY TEAMS CUSTOM APP SUPPORT

In Teams, go to left sidebar and check:
Apps -> Manage your apps -> look for "Upload a custom app"

  - If the option is missing, ask your Teams admin to enable sideloading
    (Teams Admin Center -> Setup policies -> enable "Upload custom apps")
  - If the option exists but you hit a permission error in testing the
    bot, ask your admin to allow custom app usage for your account
    (Teams Admin Center -> Permission policies -> allow custom apps)


CONFIRM PYTHON VERSION

Make sure Python 3.11 or later is installed. Python 3.11 or 3.12 is
ideal. (3.14 works as well, but some dependencies install more slowly.)
